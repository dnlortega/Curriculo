## 🚀 Como rodar este projeto

Se você for um desenvolvedor querendo explorar o código deste portfólio:

1. Clone o repositório
```bash
git clone https://github.com/dnlortega/nomedoseurepositorio.git
```

2. Instale as dependências
```bash
npm install
```

3. Inicie o servidor de desenvolvimento
```bash
npm run dev
```

4. Acesse `http://localhost:3000` no seu navegador.
